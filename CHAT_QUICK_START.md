# AI助手对话框 - 快速开始指南

## 🚀 快速开始

### 1. 发送消息

在右侧面板的 AI 助手输入框中输入问题，然后：
- 点击 **蓝色发送按钮** 📤
- 或按 **Enter** 键

### 2. 停止生成

如果 AI 回答太长或不满意：
- 点击 **红色停止按钮** 🛑
- 已生成的内容会保留

### 3. 复制内容

#### 复制整条消息
1. 鼠标悬停在 AI 回答上
2. 点击右上角的 **Copy** 按钮
3. 看到 "Copied!" 提示表示成功

#### 复制代码块
- 每个代码块右上角都有独立的 **Copy** 按钮
- 点击即可复制代码

## 📝 Markdown 格式示例

### 要求 AI 使用 Markdown

**提示词示例**：
```
请用 Markdown 格式解释 CAD 中的坐标系统，包括代码示例
```

### AI 可以返回的格式

#### ✨ 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 📋 列表
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

#### 💻 代码
````markdown
行内代码：`console.log('Hello')`

代码块：
```javascript
function greet(name) {
    return `Hello, ${name}!`;
}
```
````

#### 🔗 链接和强调
```markdown
**粗体文本**
*斜体文本*
[链接文本](https://example.com)
```

#### 💬 引用
```markdown
> 这是一段引用文本
```

## 🎯 实用技巧

### 技巧 1：要求结构化回答
```
请用 Markdown 格式列出 CAD 绘图的 5 个最佳实践
```

### 技巧 2：要求代码示例
```

## 🎨 界面说明

### 消息样式
- **你的消息**：蓝色背景，右侧对齐
- **AI 回答**：白色/灰色背景，左侧对齐，带机器人图标

### 按钮状态
| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 等待发送 | 📤 send | 蓝色 | 点击发送消息 |
| 生成中 | 🛑 stop | 红色 | 点击停止生成 |
| 思考中 | ⏳ | 灰色 | 正在加载，请稍候 |

### 代码块特点
- 深色背景，易于阅读
- 显示编程语言（如 JavaScript, Python）
- 独立的复制按钮
- 自动滚动条（内容过长时）

## ❓ 常见问题

### Q: 为什么 Markdown 没有渲染？
**A**: 确保格式正确，代码块需要用 ``` 包裹，标题需要 # 开头

### Q: 能否编辑已发送的消息？
**A**: 当前版本不支持，但可以重新发送修正后的问题

### Q: 停止生成后能恢复吗？
**A**: 不能恢复，但可以要求 AI 继续之前的话题

### Q: 复制按钮不显示？
**A**: 鼠标悬停在消息上时才会显示，确保鼠标移到消息区域

### Q: 代码块支持哪些语言？
**A**: 支持所有常见编程语言，如：
- JavaScript / TypeScript
- Python
- Java / C++ / C#
- HTML / CSS
- SQL
- 等等
