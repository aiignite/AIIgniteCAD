## 1. 项目概览 (Project Overview)

**项目名称**: AIIgniteCAD  
**定位**: 下一代 AI 驱动的智能 CAD 编辑器 (AI-Powered Intelligent CAD Editor)  
**核心理念**: "让设计更智能，让协作更简单" (Make Design Smarter, Collaboration Simpler)

**简介**:
AIIgniteCAD 是一个基于 Web 的现代化 CAD 设计平台，它不仅具备传统 CAD 的核心绘图能力，更深度集成了 Google Gemini 2.0 人工智能模型。用户可以通过自然语言与系统交互，自动生成设计图纸、优化几何结构，并支持多人实时协作。项目采用 React 19、TypeScript 和 PostgreSQL 构建，具备企业级的数据管理和离线优先的本地存储能力。

---

## 2. 核心功能介绍 (Key Features)

### 🤖 AI 智能设计助手 (AI Design Assistant)
- **自然语言绘图**: 用户只需输入 "画一个长20宽10的矩形" 或 "设计一个现代风格的客厅平面图"，AI 即可自动生成对应的 CAD 几何图形。
- **智能建议与优化**: AI 能分析当前图纸，提供设计优化建议。
- **多模型支持**: 集成 Google Gemini 2.0 Flash 模型，响应速度快，理解能力强。

### 🧱 强大的 Blocks 块系统 (Powerful Blocks System)
- **核心亮点**: 将复杂的图形组合定义为可复用的 "块" (Blocks)。
- **灵活引用**: 支持在图纸中多次引用同一个块，并进行独立的旋转、缩放和镜像变换。
- **动态更新**: 修改块的定义后，所有引用该块的实例会自动更新，极大提高了设计修改的效率。
- **嵌套支持**: 支持块的嵌套定义（块中包含块），满足复杂组件的设计需求。

### 📐 专业级 CAD 引擎 (Professional CAD Engine)
- **全格式支持**: 完美支持 DXF 文件的导入与导出，兼容 AutoCAD 等主流软件。
- **几何运算**: 内置 50+ 个核心几何函数，支持精确的捕捉、测量和构造。
- **图层管理**: 标准的图层控制系统，支持颜色、线型和可见性管理。

### 💾 混合存储架构 (Hybrid Storage Architecture)
- **离线优先**: 基于 IndexedDB 的本地存储，断网也能继续工作，数据永不丢失。
- **云端同步**: 网络恢复后，自动将本地变更同步至 PostgreSQL 企业级数据库。
- **即时协作**: 利用 WebSocket 实现多人同时编辑同一份图纸，变更实时可见。

## 3. 使用说明 (Usage Instructions)

### 快速启动
1. **环境准备**: 确保安装 Node.js 20+ 和 Docker。
2. **一键运行**: 使用 `docker-compose up -d` 命令即可启动包括前端、后端和数据库在内的完整环境。
3. **访问地址**: 浏览器打开 `http://localhost:3400` 即可开始设计。

### 基础操作指南
- **绘图**: 点击左侧工具栏选择 "直线"、"圆形" 或 "多段线" 工具在画布上绘制。
- **AI 交互**: 点击右下角的 AI 图标打开对话框，输入指令（例如："创建一个半径为50的齿轮"），点击发送。
- **块操作**: 选中一组图形 -> 右键 "创建块" -> 输入名称。之后可从右侧面板拖入该块的实例。
- **保存与导出**: 顶部菜单栏支持 "保存项目" 或 "导出为 DXF"。

### 常用快捷键
- **撤销/重做**: `Ctrl+Z` / `Ctrl+Y` (Mac: `Cmd+Z` / `Cmd+Shift+Z`)
- **删除**: `Delete` 或 `Backspace`
- **取消操作**: `Esc`
- **完成绘制**: `Enter`

## 4. 项目特点与优势 (Features & Advantages)

| 特点 | 优势说明 |
| :--- | :--- |
| **技术栈领先** | 采用 React 19.2.3 + Vite 6.2 + TypeScript 5.8，性能卓越，开发体验极佳。 |

| **AI 深度融合** | AI 不是简单的聊天机器人，而是能直接操作 CAD 图元的设计伙伴，真正提升生产力。 |
| **开源友好** | 模块化代码结构，完善的文档（`AGENTS.md`, `README.md`），易于扩展和二次开发。 |
